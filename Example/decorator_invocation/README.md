## How to Install Virtual Environments?

Download the `.tar.gz` packages from [PCARTExampleVenvs](https://drive.google.com/drive/folders/1Nl8WAJVfsLEzCaJ5P5YNIknqkqjTmXSX?usp=drive_link) or [PCARTExampleVenvs](https://pan.nuaa.edu.cn/share/451ccdb48cc752be54af6a3912) (faster in China) and install the virtual environment by executing the following commands: 
```shell
cd /home/usr/anaconda3/envs
mkdir envName
tar -xzvf envName.tar.gz -C envName
```
